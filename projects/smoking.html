Healthcare Data Analysis

흡연 여부 데이터 분석을 통한 건강 인사이트 도출

흡연 여부에 따른 건강 지표 차이를 분석하고, 데이터 기반 건강 인사이트를 도출한 프로젝트입니다.

📆 기간 2026.04.27 ~ 2026.04.29
👤 역할 데이터 전처리 / EDA / 시각화
🏷️ 유형 Exploratory Data Analysis
🛠️ 사용 기술 Python · Pandas · Matplotlib

Project Overview

흡연은 다양한 질환의 위험 요인으로 알려져 있지만, 건강검진 데이터에서는 실제 어떤 차이가 나타나는지 데이터를 기반으로 확인하고자 했습니다. 흡연 여부에 따른 주요 건강지표를 비교 분석하여 생활습관과 건강의 상관관계를 시각적으로 도출했습니다.

What I Did

🧹 데이터 전처리

결측치 처리, 이상치 확인, 중복 제거를 통해 분석에 적합한 데이터 형태로 정리했습니다.

📊 탐색적 데이터 분석

흡연 여부에 따른 건강지표 차이를 비교하고, 변수 간 상관관계를 분석했습니다.

💡 인사이트 도출

흡연자와 비흡연자의 건강 지표 차이를 바탕으로 데이터 기반 인사이트를 정리했습니다.

Analysis Results

흡연자와 비흡연자의 주요 건강 지표를 비교하여 차이가 두드러지는 항목을 정리했습니다.

🩸 헤모글로빈 증가

흡연자는 비흡연자 대비 헤모글로빈 수치가 높게 나타났습니다.

🧪 중성지방 증가

흡연자의 중성지방 수치가 상대적으로 높게 나타났습니다.

💙 HDL 감소

흡연자는 고밀도지단백(HDL) 수치가 낮게 나타나는 경향을 보였습니다.

📈 혈청 크레아티닌 차이

흡연 여부에 따라 혈청 크레아티닌 수치에서도 차이를 확인했습니다.

Retrospective

😢 아쉬웠던 점

데이터 설명이 부족해 일부 컬럼의 의미와 측정 기준을 명확하게 해석하기 어려웠습니다.

또한 기초 통계와 시각화 중심으로 진행되어 예측 모델이나 서비스 활용 방안까지 확장하지 못한 점이 아쉬웠습니다.

🚀 개선한다면

운동량, 식습관, 음주 여부 등 추가 변수를 활용하여 분석 정확도를 높이고,

흡연 여부 예측 모델과 대시보드를 구축해 실제 건강관리 서비스로 확장해 보고 싶습니다.

데이터 전처리가 분석 결과에 큰 영향을 미친다는 점을 경험했습니다.

또한 협업 과정에서 코드 표준화와 분석 결과를 이해하기 쉽게 전달하는 능력의 중요성을 배웠습니다.
