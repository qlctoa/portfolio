Project Overview

난임 환자의 시술 데이터를 분석하여 임신 성공 가능성을 예측하는 이진 분류 AI 모델을 구축하고, 예측 확률을 기반으로 환자 맞춤형 케어 등급을 제안하는 서비스를 기획했습니다.

What I Did

🧬 임상 데이터 변수 설계

임상 기록을 분석하여 주요 피처를 선정하고, 시술 유형과 배아 상태 등 임상적 맥락을 고려해 변수를 설계했습니다.

📱 케어 플랫폼 UX/UI 기획

데이터 입력부터 예측 결과 확인, 맞춤형 가이드 제공까지 이어지는 사용자 여정을 설계하고 대시보드 화면을 기획했습니다.

Model Performance

세 가지 실험 전략을 비교하여 최종 모델을 선정하고, Public Leaderboard 성능을 기반으로 최적의 모델을 선택했습니다.

🏆 Best Model

Stacking Ensemble

📊 Public AUC 0.74232 🏅 Leaderboard 4th
🛠 Technique
CatBoost LightGBM XGBoost OOF Stacking

CatBoost, LightGBM, XGBoost를 OOF 기반으로
Stacking하여 최고 Public AUC(0.74232)를 달성했습니다.

실험 전략 제출 AUC
1안 주요 부스팅 모델 비교 0.74213
🏆 2안 다중 모델 앙상블 (Stacking) 0.74232
3안 OOF + Multi-Seed 일반화 0.74231

💡 최종 선정 이유

여러 실험안을 비교한 결과, Stacking Ensemble이 가장 높은 Public AUC(0.74232)를 기록했습니다.

CatBoost, LightGBM, XGBoost의 예측 결과를 OOF 기반으로 결합하여 단일 모델보다 안정적인 일반화 성능을 확보했기 때문에 최종 모델로 선정했습니다.
